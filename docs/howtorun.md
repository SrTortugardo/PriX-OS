# How to run this OS
This guide is made for GNU/Linux-based OSES

Its easy!, first you need to clone the repo with the next command : 
```git clone https://github.com/JoyprokashDebnath/PriX-OS.git```
Then, after doing this, you need to enter the directory with
```cd PriX-OS```
Now, its time to compile!
## Dependencies
Please make sure you have 
- gcc (with 32 bits support)
- ld
- nasm
- grub
- qemu
- vncviewer
## Compilation time :D
just type ```make``` and press enter, yes, thats all!
## How to run
type ```make run``` and on another terminal ``vncviewer localhost:5900`` (make sure you have a Xorg/Wayland Window Manager/Desktop Enviroment

## and thats all!
